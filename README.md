# 👩🏻‍💻 Guilherme Lustosa

**`Engenheiro de Software em Formação`**

Me chamo **Guilherme Lustosa do Carmo**, tenho 21 anos, sou natural do Pará e atualmente resido em Belo Horizonte - MG. 

Estou no último período do curso de **Engenharia de Software**, focado em consolidar meus conhecimentos em arquitetura, princípios de desenvolvimento e boas práticas de software. 

Atuo criando projetos pessoais e desenvolvendo soluções para clientes, desenvolvendo desde landing pages até sites com mais funcionalidades e sistemas. Sou apaixonado por tecnologia desde novo e movido pela vontade constante de evoluir.

Busco me consolidar como **Desenvolvedor Full Stack**. Atualmente tenho uma base de conhecimento em tecnologias como HTML, CSS, JavaScript, TypeScript, PHP, React, Node.js, além de Git e GitHub para controle de versão.
